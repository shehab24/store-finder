== Description ==

Store Finder allows users to conveniently search and view nearby stores, providing detailed information such as addresses, contact numbers, and opening hours. With an intuitive interface, users can easily locate desired stores and access essential details for a seamless shopping experience

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Store Finder**
3. Click on the **Store Finder** to add the block

= Download & Upload: =
1. Download the **Store Finder** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`store-finder.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Store Finder** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
